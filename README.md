Schedora Desktop
================

Schedora Desktop is a cross-platform desktop calendar and meeting scheduler
built with PySide6, SQLite, and optional REST API integration for login,
subscription management, and data sync.

Features
--------
- Smart calendar views (day, week, month)
- Meeting and reminder management
- Local SQLite storage
- Optional cloud sync and subscriptions
- System-level desktop notifications

Development Quickstart
----------------------
1. Create and activate a virtualenv.
2. Install dependencies:

   pip install -r requirements.txt

3. Initialize the local database:

   python scripts/init_db.py

4. Run the app:

   python main.py
