# Reminder Sounds

Place your custom reminder sound files in this folder.

## Supported Formats
- WAV
- MP3
- AIFF
- M4A

## Usage
1. Add your sound files to this folder
2. The app will automatically detect and use them
3. Sounds will play 2-3 times when a reminder fires

## File Naming
You can name your files anything you like. The app will scan this folder for available sounds.
